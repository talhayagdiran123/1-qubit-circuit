# 1-Qubit Quantum Circuit with Qiskit

## 📌 Project Description

This project demonstrates a simple 1-qubit quantum circuit using Qiskit.

## ⚙️ What does this circuit do?

* Creates a single qubit
* Applies a Hadamard gate to create superposition
* Measures the qubit
* Displays the result as a histogram

## 🧠 Concept

In classical computing, a bit can be either 0 or 1.
In quantum computing, a qubit can be in a superposition of both states.

## ▶️ How to Run

1. Open the notebook in Google Colab
2. Run all cells
3. Observe the output histogram

## 📊 Expected Result

You will see approximately:

* 50% probability of 0
* 50% probability of 1

* Python
* Qiskit
